async automated_savings() → bool

The automated savings algorithm.

App savings amount will be distributed to moneyboxes in priority order (excepted the Overflow Moneybox). If there is a leftover that could not been distributed, the overflow moneybox will get the leftover.

Returns:

True, if distribution is done, false, if automated savings is deactivated.

Return type:

bool

Raises:

AutomatedSavingsError: is something went wrong while session transactions.
